CSS References

W3Schools is a website produced by W3 (The World Wide Web Consortium). It is a great place to find tutorials and quick references to HTML tags and CSS attributes. They also list what features are currently supported by the most popular browsers. Their sister site, W3.org contains the official CSS documentation. Click on the W3Schools logo to go to their homepage.

The Mozilla Developer Network is an official site hosted by the Mozilla Foundation. It contains official documentation for Mozilla projects and specifications as well as general references for other technologies. Just click on the image to be redirected to their reference page.

Other Useful Sites


Codecademy has a series of HTML tutorials designed to help you quickly advance from a beginning level to an intermediate level of CSS implementation. The tutorials are generally short exercises that make up a unit with a larger project at the end of the unit that will utilize all the skills practiced up to that point. This is a good place to begin if you have little or no CSS experience.

Khan Academy is very similar to Codecademy. It has simple exercises and larger projects to help you get comfortable with CSS as quickly as possible. Stylistically, some of the concepts are taught differently than at Codecademy, so check both out and see which you prefer.

Feel free to contact us with suggestions for other sites.
